# Image interpolation
Canny edge detection implemenation.

## Build project
Build project with cmake:
```
mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=Release ..
make
```

## Run project
Run the canny edge detection:
```
./canny_edge_detection <image_file>
```

For example:
```
./canny_edge_detection 00.png
```
